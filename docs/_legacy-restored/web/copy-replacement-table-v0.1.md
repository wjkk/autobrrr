## 1. 文档目标

本文档用于把研究原型中的高风险文案，替换为功能语义一致、但不直接复用原文的正式前端占位文案。

当前要求：

- 保留功能语义
- 不照搬真实页面整句文案
- 优先使用中性、可后续演进的产品语言

## 2. 页面级替换表

| 场景 | 研究原型倾向 | 正式前端占位文案 |
